### Инструменты для конвертирования xml-файлов в CONLL-u формат
с коррекцией пунктуации и дефисных написаний

### Порядок действий

Запустите RNCtoUD_pipeline.py в командной строке, указав путь к исходным данным и к папке, в которую вы хотите поместить результаты:

```
python RNCtoUD_pipeline.py(source_path, output_path)
```
В текущем решении используется токенизатор UDpipe, в частности, модель [GE20_STR-v2.udpipe](https://disk.yandex.ru/d/csuG_5rJol0_oA)

### Что умеют скрипты
1. Пунктуация в начале предложений возвращается на место
2. Перевод в conllu
3. * Разделение предложений, где было указано
   * Склеивание предложений, где есть разрывы по :
4. Замена UD-ext помет
5. Синтаксис
6. Возвращение UD-ext помет
